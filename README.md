## 项目简介

本项目旨在建立一个全面的广东省学校信息数据库，覆盖从幼儿园到大学的各教育阶段。每所学校独立建档，按教育级别分目录管理，便于查阅和维护。

### 收录范围

- 幼儿园（含公办、民办、普惠性）
- 小学
- 初中（含九年一贯制学校）
- 初中-初职（初中阶段职业教育）
- 高中（含完全中学、十二年一贯制学校）
- 高中-高职（中等职业学校、技工学校）
- 大学（本科院校）
- 大学-大专（专科院校、高等职业学校）
- 党校

### 文件命名规则

- **学校文件**：`城市-区县-学校名.md`，例如：`广州市-白云区-黄边小学.md`
- 各教育阶段目录下的 README.md 包含该级别学校的总览与索引

## 学校信息模板

每所学校的信息文件应遵循统一模板，详见 [TEMPLATE.md](TEMPLATE.md)。

### 通用字段（所有学校适用）

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 学校名称 | 当前正式名称 | 是 |
| 所在城市 | 地级市 | 是 |
| 所在区域 | 区/县 | 是 |
| 学校地址 | 详细地址 | 是 |
| 学校性质 | 公办 / 民办 / 其他 | 是 |
| 成立时间 | 建校年份 | 否 |
| 曾用名 | 历史名称，多个用逗号分隔 | 否 |
| 百度百科 | 百度百科链接 | 否 |
| 官网 | 学校官方网站 | 否 |
| 学校简介 | 100-300字简介 | 是 |
| 特色亮点 | 学校的突出优势 | 否 |

## 各阶段收录字段

除通用字段外，不同阶段的学校还需收录以下差异化信息：

### 幼儿园

| 字段 | 说明 |
|------|------|
| 办园等级 | 省一级 / 市一级 / 区一级 / 未评级 |
| 是否普惠性 | 是 / 否 |
| 收费标准 | 保教费等 |
| 办园规模 | 班级数、在园人数 |
| 对口小学 | 如有直升或对口信息 |

### 小学

| 字段 | 说明 |
|------|------|
| 学校等级 | 省一级 / 市一级 / 区一级 |
| 对口中学 | 小升初对口学校 |
| 划片范围 | 招生地段说明 |
| 学位预警 | 是否有学位紧张预警 |
| 办学规模 | 班级数、在校生人数 |

### 初中

| 字段 | 说明 |
|------|------|
| 学校等级 | 省一级 / 市一级 / 区一级 |
| 是否完全中学 | 是否含高中部 |
| 对口小学 | 划片对口的小学 |
| 中考成绩 | 近年中考平均分、高分段占比 |
| 指标生名额 | 名额分配情况 |

### 高中

| 字段 | 说明 |
|------|------|
| 学校等级 | 国家级示范性 / 省级示范性 / 市级示范性 |
| 中考录取分数线 | 近3年户籍生/非户籍生分数线 |
| 高考特控率 | 特殊类型招生控制线上线率 |
| 高考本科率 | 本科上线率 |
| 竞赛成绩 | 五大学科竞赛获奖情况 |
| 特色班型 | 重点班、实验班等信息 |

### 大学

| 字段 | 说明 |
|------|------|
| 办学层次 | 本科 / 专科 |
| 是否211 | 是 / 否 |
| 是否985 | 是 / 否 |
| 是否双一流 | 是 / 否，及一流学科 |
| 硕士点 | 一级学科硕士点数量 |
| 博士点 | 一级学科博士点数量 |
| 博士后流动站 | 数量 |
| 重点学科 | 国家级 / 省级重点学科 |
| 学科评估 | 第四/五轮学科评估结果 |
| 院士数量 | 两院院士 |
| 录取分数线 | 近年各批次录取分数 |
| 就业率 | 毕业生就业率 |
| 知名校友 | 代表性校友 |

## 数据来源与免责声明

### 数据来源

- 广东省教育厅及各地市教育局官网
- 各学校官方网站
- 百度百科、维基百科
- 阳光高考平台（教育部）
- 各地市政府信息公开数据

### 免责声明

1. 本项目所有信息**仅供参考**，不构成任何招生、择校建议。
2. 学校信息可能随时变更，请以各学校及教育主管部门**官方发布**的信息为准。
3. 本项目收录的信息可能存在滞后或不准确之处，欢迎[反馈纠正](#贡献指南)。
4. 本项目为非商业用途的公益信息整理项目。

## 贡献指南

欢迎通过以下方式参与贡献：

### 如何贡献

1. **Fork** 本仓库
2. 按照 [模板](TEMPLATE.md) 创建或更新学校信息文件
3. 确保文件命名符合规范：`区县-学校名.md`
4. 提交 **Pull Request**，说明修改内容

### 数据格式要求

- 使用 Markdown 格式
- 遵循 TEMPLATE.md 的结构
- 信息尽量填写完整，必填字段不可留空
- 数据需注明来源

### 反馈问题

如发现信息错误或遗漏，请 [提交 Issue](../../issues) 并注明：

- 学校名称及所在地区
- 需要修正的具体内容
- 可靠的信息来源链接
