## 搜索技巧参考

### 使用 Playwright 浏览器搜索
```bash
# 1. 打开 Baidu 搜索
mcp__playwright__browser_navigate url="https://www.baidu.com/s?wd=学校名+地址"

# 2. 查找页面中的地址信息
mcp__playwright__browser_find text="学校名"

# 3. 提取页面文本
mcp__playwright__browser_evaluate function="document.body.innerText.substring(0, 3000)"
```

### 使用 WebFetch 获取百度百科（需可用）
```
WebFetch url="https://baike.baidu.com/item/学校名" prompt="提取学校地址"
```

### 使用 WebSearch 搜索（需 API 正常）
```
WebSearch query="学校名 地址 官网"
```
